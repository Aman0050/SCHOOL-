# Core Agent Directives for EduXeno Project

## The EduXeno Data Trust Principle
**CRITICAL RULE FOR ALL CODE CHANGES AND DESIGNS**

Data accuracy is the highest priority in EduXeno. Before any feature is released, it must satisfy:
1. Data Integrity
2. Data Consistency
3. Auditability
4. Security
5. Reliability

Accurate data is non-negotiable. Schools will never forgive incorrect attendance, wrong fee balances, missing payments, or cross-school data leakage. Trust in data is more important than fast, beautiful, or feature-rich software.

Every single change you make to models, controllers, or logic MUST be vetted against these 5 pillars before user experience and additional functionality are considered.
